# Reading Theme-Changer with AI Assistant

Браузерне розширення для Google Chrome, яке покращує читання веб-контенту за допомогою візуальної персоналізації, налаштування тексту та AI-функцій (пояснення, спрощення, резюме та переклад).

---

## Основні можливості

<p align="center">
  <img src="screenshots/main.PNG" width="350"><br>
  <em>Головний інтерфейс</em>
</p>

- Готові теми оформлення для комфортного читання
- Гнучкі налаштування типографіки тексту
- AI-обробка виділеного тексту:
  - спрощення
  - пояснення
  - резюмування
- Автоматичний переклад сторінок
- Збереження налаштувань окремо для кожного сайту
- Швидкий доступ до AI-функцій через контекстне меню
- Робота без реєстрації та з локальним збереженням даних

---

## Установка розширення (локально)

1. Клонувати репозиторій або завантажити його як ZIP
2. Розпакувати архів у будь-яку папку
3. Відкрити в Chrome сторінку `chrome://extensions/`
4. Увімкнути **Developer mode**
5. Натиснути **Load unpacked**
6. Вибрати папку з розширенням

Після цього іконка розширення з'явиться на панелі інструментів браузера.

## Перший запуск та AI-функції

<p align="center">
  <img src="screenshots/AI_Assistant.PNG" width="350"><br>
  <em>Налаштування AI-асистента з конфігурацією API-ключа</em>
</p>

- Базові функції тем та тексту доступні одразу
- Для AI-можливостей потрібно додати API-ключ OpenRouter
- У popup-вікні відкрий секцію **AI Features**
- Введи ключ у форматі `sk-or-v1-...` та збережи його
- За замовчуванням використовується модель `deepseek/deepseek-chat-v3.1`
- За потреби можна змінити модель у файлі `background.js`, в рядку: 
  `model: 'deepseek/deepseek-chat-v3.1'` 
  на будь-яку іншу модель OpenRouter з таким самим форматом виклику

Ключ зберігається локально в `chrome.storage.local` і використовується лише для запитів до OpenRouter API.

## Налаштування тексту

<p align="center">
  <img src="screenshots/Change_Text.PNG" width="350"><br>
  <em>Контроль форматування тексту: розмір шрифту, міжрядковий інтервал, відступи та сімейство шрифтів</em>
</p>

Налаштуй параметри тексту під свої вподобання:
- Розмір шрифту (10-32px)
- Міжрядковий інтервал (1.0-2.5)
- Відступи між абзацами (0-30px)
- Вибір сімейства шрифтів

Всі налаштування автоматично зберігаються для кожного домену окремо.

## Використання AI через контекстне меню

<p align="center">
  <img src="screenshots/Explanation.PNG" width="600"><br>
  <em>Функція AI-пояснення з детальним контекстом про виділений текст</em>

2. Клікни правою кнопкою миші
3. Обери одну з опцій:

   - Пояснити текст
   - Резюмувати текст
4. Результат з'явиться у модальному вікні

## Автоматичний переклад сторінки

<p align="center">
  <img src="screenshots/Before_After.png" width="700"><br>
  <em>Процес перекладу: український текст перекладається на англійську в реальному часі</em>
</p>

- У popup-вікні обери цільову мову
- Натисни **Перекласти сторінку**
- Переклад виконується поступово блоками
- Можна читати перекладений текст під час обробки
- Прогрес перекладу відображається в нижньому правому куті
